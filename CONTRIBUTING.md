# Contributing to AI-Powered QE

We love your input! We want to make contributing to this repository as easy and transparent as possible.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-ai-tool`)
3. Commit your changes (`git commit -m 'Add amazing AI tool'`)
4. Push to the branch (`git push origin feature/amazing-ai-tool`)
5. Open a Pull Request

## Contribution Guidelines

- Follow the existing Python/TypeScript style and structure
- Add working examples where possible
- Ensure any AI tools have clear usage instructions
- Do not commit any API keys or secrets
